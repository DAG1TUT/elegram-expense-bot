# Автоответы в Instagram Direct — кофейня «На Восходе»

Сервис принимает входящие сообщения в Instagram Direct через вебхук Meta и отправляет клиентам автоматические ответы по ключевым словам (часы работы, адрес, меню, бронь, телефон и т.д.).

**Пошаговая инструкция «куда заходить и что нажимать»:** см. файл **[ИНСТРУКЦИЯ_ПОШАГОВО.md](ИНСТРУКЦИЯ_ПОШАГОВО.md)**.

## Требования

- **Instagram**: аккаунт Business или Creator, привязанный к Facebook-странице (или тестовой странице).
- **Meta for Developers**: приложение с подключённым продуктом **Instagram** → **Instagram Messaging**.
- **Права**: для продакшена нужен Advanced Access к `instagram_manage_messages` (и обычно `instagram_basic`, `pages_manage_metadata` для вебхуков).
- **Публичный URL** для вебхука (например ngrok для теста или VPS/облако для продакшена).

## Быстрый старт

### 1. Переменные окружения

Создайте файл `.env` в папке `instagram_dm_bot` (или задайте переменные в системе):

```bash
# Секретный токен для верификации вебхука (придумайте свой)
INSTAGRAM_VERIFY_TOKEN=ваш_секретный_токен_верификации

# Page Access Token страницы Facebook, привязанной к Instagram
INSTAGRAM_PAGE_ACCESS_TOKEN=ваш_page_access_token

# Необязательно: задержка перед ответом в секундах (по умолчанию 1)
# INSTAGRAM_REPLY_DELAY_SEC=1.0

# Необязательно: версия Graph API (по умолчанию v21.0)
# META_GRAPH_API_VERSION=v21.0
```

### 2. Где взять Page Access Token

1. [developers.facebook.com](https://developers.facebook.com) → Ваше приложение → **Instagram** → **Instagram Messaging**.
2. Выберите страницу, связанную с Instagram-аккаунтом кофейни.
3. Сгенерируйте **Page Access Token** с правами `instagram_manage_messages` (и при необходимости `pages_messaging`).
4. Для долгоживущего токена: через **Tools** → **Graph API Explorer** или через процесс продления токена (см. документацию Meta).

### 3. Настройка вебхука в Meta

1. В настройках приложения: **Instagram** → **Instagram Messaging** → **Webhooks** (или **Configuration** → **Webhooks**).
2. **Callback URL**: `https://ваш-домен.ru/webhook/instagram`
3. **Verify Token**: то же значение, что и `INSTAGRAM_VERIFY_TOKEN`.
4. Подпишитесь на поле **messages** (и при необходимости **messaging_postbacks** и т.д.).

### 4. Запуск

```bash
cd instagram_dm_bot
pip install -r requirements.txt
python app.py
```

По умолчанию сервер слушает порт 5000. Для теста можно использовать [ngrok](https://ngrok.com): `ngrok http 5000` и указать выданный URL в Callback URL.

### 5. Проверка

- **Верификация**: при сохранении вебхука Meta отправит GET на ваш URL — в логах должно появиться «Webhook verified».
- **Сообщения**: напишите в Direct аккаунту кофейни — должен прийти автоответ по ключевым словам (например «часы», «адрес», «меню»).

## Правила Meta (кратко)

- Отвечать можно в течение **24 часов** с момента последнего сообщения от пользователя; дальше — только шаблонные сообщения (если разрешены).
- Лимиты по количеству сообщений в час; не рассылать сообщения пользователям, которые сами не писали.
- Подробнее: [Messenger Platform Policy](https://developers.facebook.com/docs/messenger-platform/policy).

## Что отвечает бот

Ответы заданы в `replies.py`: по ключевым словам (часы, адрес, меню, бронь, телефон, доставка, привет и т.д.) отправляется готовый текст. Если фраза не распознана — отправляется подсказка с вариантами и телефоном. Тексты можно менять под себя.

## Структура

- `app.py` — Flask: верификация вебхука, приём POST, вызов логики ответа и отправка через Graph API.
- `replies.py` — словарь ключевых слов и текстов ответов для кофейни «На Восходе».
