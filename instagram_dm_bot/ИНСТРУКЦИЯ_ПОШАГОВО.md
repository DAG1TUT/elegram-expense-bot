# Пошаговая настройка: куда заходить и что нажимать

Инструкция для настройки автоответов в Instagram Direct кофейни «На Восходе».

---

## Часть 1. Подготовка Instagram и Facebook

### Шаг 1.1. Проверить тип аккаунта Instagram

1. Откройте **Instagram** (приложение или instagram.com).
2. Зайдите в аккаунт кофейни.
3. **Профиль** → иконка **☰ (меню)** → **Настройки**.
4. Внизу найдите пункт **«Аккаунт»** или **«Профессиональный аккаунт»**.
5. Если там есть **«Переключить на профессиональный аккаунт»** — переключите (выберите **Бизнес** или **Автор**).
6. Если уже написано **«Переключить на личный аккаунт»** — аккаунт уже профессиональный, всё ок.

### Шаг 1.2. Привязать Instagram к странице Facebook

1. В тех же настройках Instagram: **Настройки** → **Аккаунт** → **Поделиться в других приложениях** / **Связанные аккаунты** (или **Мета-аккаунты**).
2. Найдите пункт про **Facebook** или **Страница Facebook**.
3. Нажмите **«Подключить»** / **«Связать страницу»**.
4. Войдите в Facebook, если попросит.
5. Выберите страницу кофейни или **создайте новую страницу** (например «Кофейня На Восходе»).
6. Подтвердите связку. После этого Instagram и страница будут связаны.

Если такого пункта нет — иногда связка делается через **Facebook**:
- Зайдите на [facebook.com](https://facebook.com) → **Страницы** → ваша страница кофейни.
- **Настройки страницы** → **Instagram** → **Подключить аккаунт** и войти в Instagram кофейни.

## Часть 2. Создание приложения в Meta for Developers

### Шаг 2.1. Зайти в кабинет и зарегистрироваться как разработчик

1. Откройте в браузере: **[developers.facebook.com](https://developers.facebook.com)**.
2. Войдите в свой **Facebook-аккаунт** (лучше тот, кто админ страницы кофейни).
3. Если сайт просит **зарегистрироваться как разработчик** — нажмите **«Начать»** / **«Get Started»** и примите условия. Без этого кнопка «Создать приложение» не появится.
4. После регистрации откроется кабинет. Кнопка создания приложения может быть в разных местах (главная, меню «Мои приложения»), поэтому проще перейти по прямой ссылке (шаг 2.2).

### Шаг 2.2. Создать приложение

1. Откройте **прямую ссылку создания приложения**: **[developers.facebook.com/apps/creation/](https://developers.facebook.com/apps/creation/)**.
2. Если откроется форма: нажмите **Next** → введите **название приложения** (например «На Восходе — автоответы») и **контактный email**.
3. В разделе **Use cases** выберите **«Other»** (Другое) и нажмите **Next** (или **Create app**).
4. В шаге **Business** можно выбрать существующий бизнес-аккаунт или пропустить; нажмите **Next**.
5. Подтвердите и нажмите **Go to dashboard** — приложение создано.

**Если кнопки «Создать приложение» нет или ссылка не открывает форму:**
- Убедитесь, что вы **зарегистрированы как разработчик** (при первом заходе на developers.facebook.com нажмите «Начать» / Get Started и примите условия).
- Попробуйте: **Мои приложения** (My Apps) → вверху или сбоку может быть **«Создать приложение»** (Create App).
- У одного аккаунта может быть не больше 15 приложений; если лимит достигнут, удалите или выйдите из старых приложений.

**Если Meta пишет, что нужно «пользоваться этим устройством какое-то время» перед Get Started:**  
Это проверка безопасности. Решение: несколько дней заходите в Facebook (или в этот же браузер) с этого компьютера/телефона — заходите на facebook.com, листайте ленту, зайдите в настройки. Через 2–7 дней снова откройте [developers.facebook.com](https://developers.facebook.com) и нажмите Get Started — доступ часто открывают. Либо попробуйте с другого устройства/браузера, которым вы давно пользуетесь с этим аккаунтом.

### Шаг 2.3. Добавить продукт «Instagram»

1. После создания откроется панель приложения.
2. В списке продуктов найдите **«Instagram»** (иконка с логотипом Instagram).
3. Нажмите **«Настроить»** (Set up) напротив Instagram.
4. В выпадающем списке выберите **«Instagram Messaging»** (обмен сообщениями) и нажмите **«Настроить»**.

### Шаг 2.4. Подключить страницу и получить токен

1. В левом меню откройте **Instagram** → **Instagram Messaging** (или **Basic Display** / **Graph API** — зависит от интерфейса; нужен раздел, где есть **Instagram Messaging**).
2. Найдите блок **«Подключение к Instagram»** или **«Page Access Token»** / **«Access Token»**.
3. Нажмите **«Подключить или создать страницу»** / **«Add or remove Pages»**.
4. Выберите **страницу Facebook**, привязанную к Instagram кофейни.
5. Разрешите права (галочки), которые запросит Meta (например управление сообщениями).
6. После подключения страницы нажмите **«Generate Token»** / **«Сгенерировать токен»** рядом с этой страницей.
7. **Скопируйте токен** и сохраните в надёжное место — это ваш **Page Access Token**. Позже вставите его в `.env` как `INSTAGRAM_PAGE_ACCESS_TOKEN`.

## Часть 3. Настройка вебхука (куда Meta будет слать сообщения)

Чтобы Meta присылала вам входящие сообщения из Direct, нужен **публичный адрес** вашего сервера.

### Шаг 3.1. Запустить бота и открыть его в интернете (тест через ngrok)

**Вариант А — тест на своём компьютере через ngrok:**

1. Установите [ngrok](https://ngrok.com/download) (или зарегистрируйтесь и скачайте).
2. В терминале:
   ```bash
   cd instagram_dm_bot
   pip install -r requirements.txt
   python app.py
   ```
3. В **другом** окне терминала запустите: `ngrok http 5000`.
4. В выводе ngrok найдите строку вида **`https://xxxx-xx-xx-xx-xx.ngrok-free.app`** — это ваш временный публичный URL. Скопируйте его.

**Вариант Б — если бот уже на сервере (VPS, хостинг):**  
У вас уже должен быть адрес вида `https://ваш-домен.ru`. Используйте его.

### Шаг 3.2. Придумать Verify Token

1. Придумайте любую секретную строку, например: `navoskhod_coffee_2025`.
2. Запомните или запишите — это **Verify Token**. Такое же значение потом укажете в `.env` как `INSTAGRAM_VERIFY_TOKEN`.

### Шаг 3.3. Указать вебхук в настройках приложения Meta

1. Снова откройте **[developers.facebook.com](https://developers.facebook.com)** → **Мои приложения** → выберите ваше приложение (например «На Восходе — автоответы»).
2. В левом меню откройте **Instagram** → **Instagram Messaging** (или **Configuration** / **Настройка**).
3. Найдите раздел **«Webhooks»** / **«Вебхуки»**.
4. Нажмите **«Add Callback URL»** / **«Добавить URL обратного вызова»** (или **«Edit»** / **«Настроить»** у существующего).
5. Заполните:
   - **Callback URL**:  
     `https://ВАШ-URL/webhook/instagram`  
     Например: `https://xxxx.ngrok-free.app/webhook/instagram` или `https://ваш-домен.ru/webhook/instagram`.
   - **Verify Token**: вставьте вашу секретную строку (например `navoskhod_coffee_2025`).
6. Нажмите **«Verify and Save»** / **«Проверить и сохранить»**.
7. В этот момент Meta отправит запрос на ваш сервер. Убедитесь, что **бот запущен** и **ngrok включён**. Если всё верно, вебхук сохранится без ошибки.
8. Подпишитесь на поля:
   - Отметьте **«messages»** (сообщения).
   - При необходимости **«messaging_postbacks»** (нажатия кнопок).
9. Сохраните изменения.

### Шаг 3.4. Подключить Instagram-аккаунт к приложению

1. В том же приложении найдите раздел **«Instagram»** или **«Instagram Messaging»**.
2. Должен быть пункт вроде **«Subscribe Instagram account to app»** / **«Подписать аккаунт на приложение»** или список страниц.
3. Убедитесь, что выбрана **страница**, связанная с Instagram кофейни, и что подписка на вебхуки для этой страницы активна (часто это делается автоматически при сохранении Callback URL).

## Часть 4. Файл .env и запуск

### Шаг 4.1. Создать файл .env

1. Откройте папку **`instagram_dm_bot`** в проекте.
2. Создайте файл с именем **`.env`** (точка в начале, без расширения).
3. Вставьте (подставьте свои значения):

```env
INSTAGRAM_VERIFY_TOKEN=navoskhod_coffee_2025
INSTAGRAM_PAGE_ACCESS_TOKEN=сюда_вставьте_скопированный_page_access_token
```

Токен верификации должен **совпадать** с тем, что вы ввели в настройках вебхука в шаге 3.3.

### Шаг 4.2. Запустить бота

В терминале:

```bash
cd instagram_dm_bot
python app.py
```

Должно появиться что-то вроде: `Running on http://0.0.0.0:5000`. Оставьте окно открытым (или запустите на сервере через systemd/supervisor).

## Часть 5. Проверка

1. С **другого** аккаунта Instagram (или с тестового) напишите в **Direct** аккаунту кофейни любое сообщение, например: **«часы»** или **«адрес»**.
2. В течение нескольких секунд должен прийти автоматический ответ с часами работы или адресом.
3. Если ответа нет — проверьте:
   - бот запущен;
   - в логах бота нет ошибок;
   - Callback URL доступен из интернета (ngrok не закрыт);
   - Page Access Token скопирован полностью и страница привязана к Instagram.

## Краткий чек-лист

| № | Действие |
|---|----------|
| 1 | Instagram кофейни — профессиональный аккаунт (Бизнес/Автор) |
| 2 | Instagram привязан к странице Facebook |
| 3 | developers.facebook.com → Создать приложение → добавить продукт Instagram (Instagram Messaging) |
| 4 | Подключить страницу кофейни и сгенерировать Page Access Token → сохранить в `.env` |
| 5 | Запустить бота (`python app.py`), при тесте — запустить ngrok (`ngrok http 5000`) |
| 6 | В приложении Meta: Instagram → Webhooks → Callback URL = `https://ВАШ-URL/webhook/instagram`, Verify Token = как в `.env` → Verify and Save |
| 7 | Подписаться на поле **messages** |
| 8 | В `.env` указать `INSTAGRAM_VERIFY_TOKEN` и `INSTAGRAM_PAGE_ACCESS_TOKEN` |
| 9 | Написать в Direct кофейне «часы» или «адрес» и проверить автоответ |

Если на каком-то шаге интерфейс Meta выглядит иначе (например, пункты на английском), ориентируйтесь по словам **Webhooks**, **Callback URL**, **Verify Token**, **Page Access Token**, **Instagram Messaging** — они обычно совпадают.
