# Как сделать сайт доступным для всех

Сайт готов к публикации. Ниже — два простых бесплатных способа.

---

## Вариант 1: Netlify (самый быстрый, без регистрации в Git)

1. Зайди на **https://app.netlify.com** и зарегистрируйся (email или через Google/GitHub).
2. Перейди в **https://app.netlify.com/drop**.
3. Открой на компьютере папку **`coffee-shop-website`** (там лежат `index.html`, `styles.css`, `script.js`, папка `images`).
4. Перетащи эту папку в окно браузера на странице Netlify Drop (в область «Drop your site here»).
5. Через несколько секунд появится ссылка вида **`название-сайта-123.netlify.app`** — это и есть твой рабочий сайт. Её можно скидывать клиентам.

**Свой домен (например `navoshode.ru`):** в кабинете Netlify: Site settings → Domain management → Add custom domain.

---

## Вариант 2: GitHub Pages (бесплатно, удобно, если пользуешься GitHub)

1. Создай на **https://github.com** новый репозиторий (например `navoshode`). Не добавляй README при создании.
2. На компьютере открой терминал и перейди в папку с сайтом:
   ```bash
   cd путь/к/coffee-shop-website
   ```
3. Выполни по очереди:
   ```bash
   git init
   git add .
   git commit -m "Сайт кофейни На Восходе"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/navoshode.git
   git push -u origin main
   ```
   (подставь свой логин GitHub и имя репозитория вместо `navoshode`).
4. На GitHub открой репозиторий → **Settings** → **Pages**.
5. В разделе **Source** выбери **Deploy from a branch**.
6. В **Branch** выбери **main** и папку **/ (root)**. Сохрани (**Save**).
7. Через 1–2 минуты сайт откроется по адресу: **`https://ТВОЙ_ЛОГИН.github.io/navoshode`**.

**Свой домен:** в тех же настройках Pages укажи свой домен в поле **Custom domain**.

---

## Что нужно заливать в интернет

В сеть должны попасть только файлы из папки **`coffee-shop-website`**:

- `index.html`
- `styles.css`
- `script.js`
- папка `images` со всеми файлами внутри (logo.png, about-interior.jpg и т.д.)

Файл `logo.pdf` можно не заливать — на сайте используется `logo.png`.

## Свой домен (например navoshode.ru)

### Шаг 1: Купи домен

Зарегистрируй домен у любого регистратора:
- **Россия:** Reg.ru, Timeweb, Beget, Ru-Center, nic.ru
- **Зарубежные:** Cloudflare (часто дёшево), Namecheap, Google Domains

Выбери имя (например **navoshode.ru** или **na-voshode.ru**) и оплати. После оплаты у тебя будет доступ в «личный кабинет» к настройкам DNS этого домена.

---

### Шаг 2: Опубликуй сайт на Netlify

1. Зайди на **https://app.netlify.com/drop**.
2. Перетащи папку **coffee-shop-website** в окно браузера.
3. Дождись появления ссылки вида `что-то.netlify.app` — сайт уже в сети.

---

### Шаг 3: Подключи домен к Netlify

1. В Netlify открой свой сайт → **Domain settings** (или **Site configuration** → **Domain management**).
2. Нажми **Add custom domain** или **Add domain alias**.
3. Введи свой домен (например **navoshode.ru**) и сохрани.
4. Netlify покажет, что нужно прописать у регистратора:
   - Обычно для **navoshode.ru** просят добавить **A-записи**:
     - Имя: `@` (или пусто, или «корень»)
     - Значение: `75.2.60.5` (IP Netlify — актуальный смотри в Netlify)
   - Или **CNAME** для поддомена (например www):
     - Имя: `www`
     - Значение: `твой-сайт.netlify.app`
5. Часто Netlify сам подставляет нужные IP — просто скопируй их из подсказки в кабинете Netlify.

---

### Шаг 4: Пропиши DNS у регистратора

1. Зайди в личный кабинет того сервиса, где купил домен (Reg.ru, Timeweb, Beget и т.д.).
2. Найди раздел **«Управление доменом»** → **«DNS-записи»** / **«Зона DNS»**.
3. Добавь записи **точно так**, как написал Netlify (A и/или CNAME). Старые A-записи для этого домена лучше удалить или заменить на те, что дал Netlify.
4. Сохрани изменения.

---

### Шаг 5: Дождись обновления DNS

Обновление DNS занимает от 5–10 минут до 24–48 часов. Когда DNS обновится, Netlify «увидит» домен и включит для него бесплатный SSL (сайт будет открываться по **https://navoshode.ru**).

В Netlify можно включить **«HTTPS»** и **«Force HTTPS»**, чтобы всех перенаправлять на защищённое соединение.

---

### Кратко

| Где | Что делать |
|-----|------------|
| **Регистратор (Reg.ru и т.п.)** | Купить домен, в DNS добавить A/CNAME из Netlify |
| **Netlify** | Опубликовать сайт (Drop), в Domain settings добавить свой домен и вписать те записи, что просит Netlify |

Если напишешь, у какого регистратора купил домен (Reg.ru, Timeweb, Beget), могу расписать точнее: куда зайти и какие кнопки нажать.
