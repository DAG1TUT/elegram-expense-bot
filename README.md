# Telegram-бот для учёта расходов

Бот записывает траты по вашим сообщениям и сам распределяет их по категориям (еда, транспорт, кафе и т.д.).

## Установка

1. Создай бота в Telegram через [@BotFather](https://t.me/BotFather), получи токен.
2. Установи зависимости (лучше через виртуальное окружение — тогда работают и голосовые):

```bash
cd telegram-expense-bot
python3 -m venv venv
./venv/bin/pip install -r requirements.txt
```

3. Запусти бота, указав токен:

**Через виртуальное окружение (рекомендуется, для голоса):**
```bash
cd telegram-expense-bot
TELEGRAM_BOT_TOKEN="твой_токен" ./venv/bin/python bot.py
```
или:
```bash
TELEGRAM_BOT_TOKEN="твой_токен" ./run.sh
```

**Без venv (голосовые не будут работать, если пакеты не ставились в системный Python):**
```bash
TELEGRAM_BOT_TOKEN="твой_токен" python3 bot.py
```

## Как пользоваться

- Напиши боту, **на что** и **сколько** потратил, например:
  - `кофе 350`
  - `500 обед в кафе`
  - `продукты 2000 руб`
  - `150 такси`
- Бот сохранит трату и подставит категорию (Еда и продукты, Кафе, Транспорт и т.д.).
- Можно отправить **голосовое сообщение**, например: «кофе пятьсот рублей» — бот распознает речь и запишет трату. Для этого нужны зависимости из `requirements.txt` и **ffmpeg** в системе (для конвертации аудио).

## Команды

| Команда | Описание |
|--------|----------|
| `/start` | Приветствие и подсказки |
| `/stats` | Траты по категориям за последние 30 дней |
| `/list` | Последние 15 трат |
| `/month` | Общая сумма за последние 30 дней |

## Категории

Категория выбирается по ключевым словам в описании. Примеры:

- **Еда и продукты** — продукты, магазин, супермаркет, молоко, хлеб…
- **Кафе и рестораны** — кофе, обед, кафе, доставка, пицца…
- **Транспорт** — такси, метро, бензин, каршеринг…
- **Дом и коммуналка** — коммуналка, интернет, аренда…
- **Покупки** — одежда, техника, маркетплейс…
- **Здоровье** — аптека, врач, спортзал…
- **Развлечения** — кино, подписка, игры…
- И другие (см. `categories.py`). Всё остальное попадает в «Прочее».

Данные хранятся в файле `expenses.db` в папке бота (SQLite).
